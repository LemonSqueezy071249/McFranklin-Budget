Google Sheets Connection

Setup guide (one-time, ~10 minutes):
1. Go to sheets.google.com and create a new spreadsheet. Name it "Family Budget".
2. In the spreadsheet, click Extensions → Apps Script.
3. Delete any existing code, then paste in the Apps Script code below and click Save.
4. Click Deploy → New Deployment. Set type to "Web app". Set "Who has access" to "Anyone". Click Deploy.
5. Copy the Web App URL it gives you and paste it below.
6. The first time you run it, Google will ask you to authorize — click through.
